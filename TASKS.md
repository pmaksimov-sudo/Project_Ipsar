## Приоритет: Средний

### 4. Реальные фотографии оборудования

Заказчик должен предоставить фотографии. Заменить CSS-заглушки в:

- [ ] `Hero` — фото или 3D-рендер системы справа (вместо карточки со статистикой или рядом)
- [ ] `Solution` — фото реального оборудования в лаборатории
- [ ] `Configurations` — фото каждой из трёх конфигураций

После добавления изображений:
- [ ] Добавить `loading="lazy"` для всех изображений ниже первого экрана
- [ ] Добавить атрибут `alt` с описанием для каждого изображения
- [ ] Оптимизировать под WebP формат
